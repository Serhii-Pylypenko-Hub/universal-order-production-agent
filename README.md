# Universal Order Production Agent

Репозиторій з локальним MVP AI-асистента для прийому замовлень, Telegram-бота, тестового web-кабінету, демо-даних і менеджерських сценаріїв.

## Де основний проєкт

Основний код зараз знаходиться у вкладеній папці:

```text
universal-order-production-agent-full-functions-1/universal-order-production-agent-full-functions
```

У цій папці лежать:

- `README.md` - повний опис проєкту;
- `BOT_TEST_INSTRUCTION_UA.txt` - інструкція для тестувальників;
- `START_APP.bat` - запуск web-кабінету;
- `START_TELEGRAM_BOT.bat` - запуск Telegram-бота;
- `documentation/` - робоча документація;
- `app/`, `templates/`, `data/`, `tests/` - код, демо-дані та перевірки.

## Швидкий старт

1. Відкрийте папку:

```text
universal-order-production-agent-full-functions-1/universal-order-production-agent-full-functions
```

2. Прочитайте:

```text
BOT_TEST_INSTRUCTION_UA.txt
```

3. Для запуску web-кабінету двічі натисніть:

```text
START_APP.bat
```

4. Для запуску Telegram-бота після налаштувань двічі натисніть:

```text
START_TELEGRAM_BOT.bat
```

## Статус

Це MVP/demo-версія для локального тестування. Вона ще не є production-розгортанням в інтернеті.
