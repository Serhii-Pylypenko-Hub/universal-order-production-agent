## Поточний фокус

Зараз система переходить від набору сервісних функцій до робочого веб-кабінету:

1. Користувач відкриває лінк.
2. Реєструється або входить.
3. Підключає Telegram, AI, Google Sheets/Calendar або лишається в local mode.
4. Запускає workspace setup.
5. Бачить базовий dashboard стану системи.

Вся бізнес-логіка має залишатися в сервісах `app/js/*`, а веб-інтерфейс має тільки викликати ці сервіси через тонкий API-шар.
