# Telegram cakes demo

## Що перевіряти в демо

Клієнт пише боту звичайним текстом українською. Бот працює тільки з темою тортів, збирає продукт, вагу або кількість, дату, побажання, алергії та зміни рецепта.

Після створення замовлення клієнт отримує підтвердження, а менеджер отримує окреме Telegram-сповіщення з номером замовлення, контактом клієнта, датою, сумою, статусом і позиціями.

Приклади для клієнта:

```text
Хочу шоколадний торт 2 кг на завтра, більше шоколаду і без горіхів
```

```text
Потрібен Berry Cheesecake 1 кг на 20 число, більше ягід
```

```text
Хочу Napoleon Cake 2 кг на суботу з написом Happy Birthday
```

```text
Хочу, щоб менеджер передзвонив мені
```

## Зворотний зв'язок

Якщо клієнт просить передзвонити, зв'язатися з менеджером або пише "callback", бот:

- створює handoff-запит;
- очищає поточну розмову;
- повідомляє клієнту, що запит передано менеджеру;
- надсилає менеджеру alert у Telegram.

## Додаткові побажання до рецепта

Для Chocolate Cake демо підтримує:

- `Extra chocolate` - більше шоколаду, додає Cocoa у розрахунок;
- `Add nuts` - додати горіхи, додає Nuts у розрахунок;
- `Remove nuts` - не класти горіхи, прибирає Nuts із рецепта;
- `Add raspberry` - додати малину;
- `Add inscription` - напис на торті.

Ці зміни проходять через `CustomizationService`, тому впливають на BOM, собівартість, ціну, години роботи та залишки.

## Команди менеджера

Подивитися замовлення:

```text
/orders
```

Подивитися конкретне замовлення:

```text
/order ORD-XXXX
```

Подивитися продукти:

```text
/products
```

Подивитися матеріали:

```text
/components
```

Подивитися рецепт і залишки по рецепту:

```text
/recipe Chocolate Cake
```

Створити знижку від суми:

```text
/discount_over_amount 1500 10
```

Це дає 10% знижки для замовлень від 1500 грн.

Створити знижку на кожне N-те замовлення:

```text
/discount_every_n 10 15
```

Це дає 15% на кожне 10-те замовлення.

Подивитися всі знижки:

```text
/discounts
```

Вимкнути знижку:

```text
/discount_disable DISC-XXXX
```

Поставити власну ціну менеджера:

```text
/set_order_price ORD-XXXX 1200 індивідуальна домовленість
```

Команда оновлює `proposed_price`, `final_price`, `pricing_source` і статус price review.

## Обмеження демо

Поки немає окремої кнопки в UI для керування знижками. Для демо це робиться командами менеджера в Telegram. Правила знижок записуються в `DiscountRules` і застосовуються під час створення наступних замовлень.
