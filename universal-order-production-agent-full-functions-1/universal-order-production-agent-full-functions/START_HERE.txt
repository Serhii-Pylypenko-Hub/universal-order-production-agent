AI Operations Local App
=======================

Це локальна версія застосунку для Windows.
Вам не потрібно знати програмування.

ЯК ЗАПУСТИТИ
------------

1. Розпакуйте архів у будь-яку папку.

2. Відкрийте розпаковану папку.

3. Двічі клікніть файл:

   start-local.bat

4. Зачекайте кілька секунд.

5. Браузер має відкритися автоматично.
   Якщо не відкрився, відкрийте браузер вручну і введіть:

   http://localhost:3000

6. Зареєструйте користувача:
   - ім'я
   - пошта
   - пароль

7. Для тесту без реальних ключів натисніть:

   Demo cakes


ЯК ЗАПУСТИТИ TELEGRAM-БОТА
--------------------------

1. У веб-кабінеті заповніть і збережіть:
   - Telegram bot token
   - Manager chat ID
   - OpenRouter API key

2. Натисніть "Запустити демо".

3. Двічі клікніть:

   start-bot-local.bat

4. Відкрийте свого Telegram-бота і напишіть замовлення.

Приклад:

   Хочу шоколадний торт 2 кг на завтра, без горіхів

Зупинити бота:

   stop-bot-local.bat


ЯК ЗУПИНИТИ
-----------

Коли завершили роботу, двічі клікніть:

   stop-local.bat


ДЕ ЗБЕРІГАЮТЬСЯ ДАНІ
--------------------

Локальні дані зберігаються всередині папки застосунку:

   data/local_workspace.json

Користувачі та сесії:

   data/web_auth.json

Логи запуску:

   .local/server.out.log
   .local/server.err.log
   .local/bot.out.log
   .local/bot.err.log


ЯКЩО НЕ ЗАПУСКАЄТЬСЯ
--------------------

1. Переконайтесь, що файл start-local.bat лежить у цій самій папці.
2. Перевірте, що папка .tools існує.
3. Перевірте, що порт 3000 не зайнятий іншою програмою.
4. Відкрийте файл .local/server.err.log і передайте його технічному спеціалісту.


ВАЖЛИВО
-------

Це локальний застосунок. Він працює на вашому комп'ютері.
Це не сайт в інтернеті, доки його окремо не розгорнули на сервері.
