AI Operations Local App
=======================

Це локальна версія застосунку для Windows.
Вам не потрібно знати програмування.


ЯК ЗАПУСТИТИ ЗАСТОСУНОК
-----------------------

1. Розпакуйте архів у будь-яку папку.
2. Відкрийте розпаковану папку.
3. Двічі клікніть файл:

   START_APP.bat

   Якщо у вашій збірці файл називається start-local.bat, відкрийте його.

4. Зачекайте кілька секунд.
5. Браузер має відкритися автоматично.
   Якщо не відкрився, відкрийте браузер вручну і введіть:

   http://localhost:3000

6. Зареєструйте користувача.
7. Для тесту без реальних ключів натисніть:

   Demo cakes


ЯК ЗАПУСТИТИ TELEGRAM-БОТА
--------------------------

1. У web-кабінеті заповніть і збережіть:
   - Telegram bot token
   - Manager chat ID
   - OpenRouter API key

2. Натисніть:

   Demo cakes

3. Відкрийте:

   Операції -> Керування ботом

4. Натисніть:

   Запустити бота

Для постійної роботи натисніть:

   Увімкнути постійно

5. Статус має показати:

   Працює · PID ...

6. Відкрийте свого Telegram-бота і напишіть:

   /start

Приклад замовлення:

   Хочу шоколадний торт 2 кг на завтра, без горіхів


ЯК ЗУПИНИТИ TELEGRAM-БОТА
-------------------------

У web-кабінеті відкрийте:

   Операції -> Керування ботом

Натисніть:

   Зупинити

Запасний спосіб, якщо web-кнопка недоступна:

   START_TELEGRAM_BOT.bat
   STOP_TELEGRAM_BOT.bat

або у старій збірці:

   start-bot-local.bat
   stop-bot-local.bat


ЯК ЗРОБИТИ БОТА ПОСТІЙНИМ
-------------------------

Якщо бот має працювати постійно і запускатися сам після входу в Windows:

1. Спочатку заповніть token/chat id/API key у web і натисніть Save.
2. Перевірте, що бот запускається кнопкою `Запустити бота`.
3. У папці застосунку двічі натисніть:

   install-bot-autostart.bat

Це створить фонову задачу Windows:

   AIOperationsTelegramBot

Після цього бот буде запускатися автоматично при вході цього Windows-користувача в систему. Якщо процес бота випадково зупиниться, watchdog перевірить це і підніме його знову.

Щоб вимкнути постійний автозапуск, натисніть у web:

   Вимкнути постійно

Запасний спосіб:

   uninstall-bot-autostart.bat


ЩО МОЖНА ПРОТЕСТУВАТИ
---------------------

У demo можна пройти повний цикл:

   матеріали -> закупка -> замовлення -> резерв -> виробництво -> списання -> видача -> залишки -> звіти

Основні місця в web:

- Огляд: demo-товари, залишки, замовлення, стан системи.
- Склад: матеріали, партії, резерви, план закупівель, звіти.
- Виробництво: взяти замовлення в роботу, завершити і списати матеріали.
- Операції: продукти, техкарти, закупки, керування ботом.
- AI: текстові й голосові команди.
- Платежі: planned-блок, поки неактивний.


ДЕ ЗБЕРІГАЮТЬСЯ ДАНІ
--------------------

Локальні дані:

   data/local_workspace.json

Користувачі та сесії:

   data/web_auth.json

Логи запуску:

   .local/server.out.log
   .local/server.err.log
   .local/bot.out.log
   .local/bot.err.log
   .local/bot-watchdog.log


ЯКЩО НЕ ЗАПУСКАЄТЬСЯ
--------------------

1. Переконайтесь, що START_APP.bat лежить у цій самій папці.
2. Перевірте, що папка .tools існує.
3. Перевірте, що порт 3000 не зайнятий іншою програмою.
4. Якщо бот не стартує з web, перевірте token, Manager chat ID і OpenRouter API key.
5. Відкрийте файл .local/server.err.log або .local/bot.err.log і передайте його технічному спеціалісту.
